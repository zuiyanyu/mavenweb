糖醋排骨


菜名：糖醋排骨
市场价格：26
会员价格：24
配料：排骨、糖、醋
菜品类型：炒菜
做法：
1. 晾干水份备用；
2. 锅内倒少量油，烧热之后，爆香姜片；
3. 放入排骨，一直煸炒到排骨变色后，表面金黄微焦；
4. 此时就可以放入黄金比例中的调料了，顺序是：先放一汤勺料酒，接着两汤勺酱油，三汤勺米醋最后四汤勺白糖，炒匀；
5. 再倒入能没过排骨的开水，调中小火焖20分钟；
6. 20分钟后调入盐，开大火收汁，收到汁浓色亮时，撒入芝麻点缀即可出锅（锅里剩下的姜片丢掉不用，最后大火收汁时注意多翻动锅内的排骨，避免烧焦哟！）。
